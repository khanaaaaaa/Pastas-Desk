# Pastas-Desk

Demo: https://khanaaaaaa.github.io/Pastas-Desk/

# What It Is

A productivity table where you have a Pomodoro timer on the table and notes to focus on what task you want to complete with lofi music.

# How I Made It

Tech Stack:

- HTML
- CSS
- JavaScript

Was some simple designing so just watched some simple clock designs on YouTube and decided to go with the round one. Also decided to go with a old design I previously did for the pomodoro timer.

# Why I Made It

Wanted to do my best to recreate a cool deskspace but also add some features along with it and saw other projects like a productive webspace being created so made one of my own. 

# What I Struggled With And What I Learnt

Wanted to make the whole thing pink but the difference between the wall and the desk wasn't visible so had to decide on a new color theme and make every feature match the same color theme. Also struggled with making the clock, notes and all not go to the desk section and only stay on the wall and learnt the use of getBoundingClientRect().
